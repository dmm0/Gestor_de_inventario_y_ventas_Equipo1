# Gestor_de_inventario_y_ventas_Equipo1

- Nombre del sistema: Gestor de Inventario y Venta(POS) para una papelería de nombre “Triates Papelería”.
Integrantes: Johana Lisset Hernández Briones, Jazmín Alejandra García Álvarez, Dante Mexicano Martínez
- Descripción del proyecto: El presente proyecto consiste en el desarrollo de un sistema informático integral de gestión administrativa, ventas e inventario para Triates Papelería, cuyo objetivo es centralizar, automatizar y optimizar los procesos operativos del negocio. Actualmente, muchas de estas actividades se realizan de forma manual, lo que puede generar errores en el control de existencias, retrasos en la atención al cliente y dificultad para consultar información histórica.
El sistema permitirá registrar ventas de manera rápida y precisa, ingresando productos, cantidades y montos totales, realizando de forma automática la actualización del inventario para mantener un control en tiempo real de las existencias. Asimismo, contará con funcionalidades para la generación de documentos comerciales, tales como notas de remisión, cotizaciones y comprobantes, los cuales podrán ser guardados, impresos o exportados en formato PDF, facilitando la documentación y seguimiento de las operaciones.
Además, el sistema incorporará un módulo para la captura y gestión de datos de clientes, incluyendo información personal y fiscal necesaria para procesos de facturación. También permitirá consultar información previa de los clientes, mejorando la atención y agilizando futuros registros.
El proyecto contempla la gestión completa del inventario, permitiendo el alta de nuevos productos, el registro de resurtidos, la actualización de precios y la consulta de existencias
mediante búsquedas por nombre. De igual manera, se incluirá un módulo para el registro de egresos, como compras de mercancía y sueldos, generando los comprobantes
correspondientes.
Para garantizar un uso seguro y organizado del sistema, se implementará un esquema de
administración de usuarios, con control de acceso mediante usuario y contraseña,
asignación de roles y restricción de funciones según el perfil del usuario. Finalmente, el
sistema contará con copias de seguridad automáticas, asegurando la protección y
recuperación de la información ante cualquier eventualidad.
En conjunto, este sistema busca mejorar la eficiencia operativa, reducir errores
administrativos y fortalecer la toma de decisiones, contribuyendo al crecimiento y
organización de Triates Papelería.
